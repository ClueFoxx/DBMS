# DBMS Experiments

This repository contains **Database Management System (DBMS) experiments** completed as part of an academic coursework.
It demonstrates practical understanding of database concepts such as **SQL queries, relational schema design, and data operations**.

## 📚 About the Project

The purpose of this repository is to document various DBMS experiments and showcase how database concepts are implemented using SQL.

Each experiment focuses on different database operations such as:

* Table creation
* Data insertion
* Query execution
* Data retrieval and manipulation

## 📂 Repository Structure

```
DBMS/
│
├── Experiment1.md
├── Experiment2.md
├── Experiment3.md
├── Experiment4.md
└── Experiment5.md
```

Each file contains the **problem statement, SQL queries, and explanation of results** for the respective experiment.

## 🛠 Technologies Used

* SQL
* Relational Database Concepts
* DBMS (MySQL / PostgreSQL / SQLite)

## 🎯 Learning Outcomes 

Through these experiments, the following concepts are explored:

* Database schema design
* SQL query writing
* Data manipulation (INSERT, UPDATE, DELETE)
* Data retrieval using SELECT queries
* Understanding relational database structures
